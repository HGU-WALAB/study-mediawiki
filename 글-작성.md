# 위키 문서 작성 방법

## URL을 통한 문서 작성

1. URL 입력창에 작성하고자 하는 주제의 제목 입력

- 예시: `/wiki/index.php/<문서제목>`
- 문서가 존재하지 않는 경우, `생성` 링크를 통해 문서를 생성할 수 있음

![](/imgs/글-작성-1.png)

![](/imgs/글-작성-2.png)

## 검색을 통한 문서 작성

1. 상단 검색창에 작성하고자 하는 주제의 제목 입력
2. 검색 결과 중 빨간 글씨로 된 주제의 제목 클릭 후 작성

![](/imgs/글-작성-3.png)

## 링크를 통한 문서 작성

1. 문서 편집에서 `[[<문서제목>]]` 형식으로 작성
2. 빨간 글씨로 된 링크 클릭 후 문서 생성

![](/imgs/글-작성-4.png)
