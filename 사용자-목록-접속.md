# 위키 사용자 목록 접속 방법

## URL을 통한 사용자 목록 접속

1. URL 입력창 뒤에 `Special:ListUsers` 입력 후 엔터

- 예시: `/wiki/index.php/Special:ListUsers`

![](/imgs/사용자-목록-접속-1.png)

## 버튼을 이용한 사용자 목록 접속

1. 왼쪽 사이드바에서 `특수 문서 목록` 버튼 클릭

![](/imgs/사용자-목록-접속-2.png)

2. `사용자와 권한` - `사용자 목록` 클릭

![](/imgs/사용자-목록-접속-3.png)
